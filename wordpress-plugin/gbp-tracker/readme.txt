Integrate your WordPress site with Track and Boost for location audits and post synchronization.

== Description ==

This plugin provides seamless integration between your WordPress site and Track and Boost, allowing you to:

* Add the Track and Boost audit widget to any page using a shortcode
* Automatically sync posts from Track and Boost to your WordPress site
* Maintain consistent branding across platforms
* Track your Google Business Profile performance

= Features =

* Easy-to-use shortcode for embedding the audit widget
* Automatic post synchronization
* Customizable widget themes
* Secure API integration
* Performance optimized

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/track-and-boost` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Track and Boost screen to configure the plugin
4. Add your API key and Location ID from your Track and Boost account
5. Use the shortcode [gbp_audit_widget] to add the widget to any page

== Frequently Asked Questions ==

= How do I get an API key? =

You can get your API key from your Track and Boost account settings page at https://trackandboost.com/settings.

= Can I customize the widget appearance? =

Yes, you can choose between light and dark themes using the theme attribute in the shortcode.
